Golang 学习笔记 01 安装及搭建开发环境(macOS GoLand)

发表于 2020-03-29 | 分类于 Golang ， 学习笔记 | | 阅读次数

前言絮语：很久很久以前，就有学习 Go 语言的计划，但是因为各种各样的问题及懒，一直拖到现在才真正开始着手，铭记为鉴！

另：本笔记所使用环境为 macOS 和 GoLand，对其他操作系统和集成开发环境未必有参考价值！

安装

Golang 官方网站
Golang 官方镜像站(国内推荐)

使用 Homebrew 查看 Go 语言信息 brew info golang，可知当前支持的最新版本 go: stable 1.14 (bottled), HEAD，操作系统版本需满足 Required: macOS >= 10.11 ✔，我这都是满足的，正式开始安装。

执行

等待安装完成后，执行 go version 验证安装情况，如果安装成功会输出 Go 语言版本信息，如：go version go1.14 darwin/amd64，如果失败重新执行 Homebrew 的安装命令即可。

配置

执行命令 go env 查看 Go 语言环境信息

GOPROXY 因为特殊的原因，要把 https://proxy.golang.org 改成比较友好的地址，推荐如下：

  • https://goproxy.io
  • https://goproxy.cn

这两个都是支持谷歌官方 sum.golang.org 的代理。

配置命令

配置后依然使用 go env 查看 Go 语言环境信息，检查配置是否生效。

GoLand 创建第一个项目

新建项目

打开 GoLand 应用，选择 New Project，在打开的窗口左侧选择 Go Modules(vgo)，右侧 Location 里选择或填写项目路径及名称后点击Create，创建一个空的 Go 语言项目。
