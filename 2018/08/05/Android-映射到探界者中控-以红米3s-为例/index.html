Android 映射到探界者中控[以红米3s 为例]

发表于 2018-08-05 | 分类于 汽车 | | 阅读次数

解决方案

本方案以红米3s手机为例，小米手机与此类似；其他厂商的手机可以先看看有没有 Google 服务框架，有 Google 服务框架比较好弄，直接 root，再翻墙安装 Android Auto，配置好即可；没有 Google 服务框架的要么找官方的国际版 ROM 刷机，要么就刷第三方的原生 Android 的 ROM，然后再 root，再翻墙安装 Android Auto。从 root 起，到后边的配置，各种手机基本上是通用的。好了，闲话不多说，准备开搞，搞这个的首要条件是要有耐心和信心，次要条件还是要有耐心和信心，正所谓生命不息折腾不止。

理论上此方案适用于所有支持 Android Auto 的汽车。

刷 MIUI 国际版

小米的手机，国内版的 MIUI 即使自己安装了 Google 服务框架也无法正常映射（已折腾 n 多次无解），所以不用纠结直接刷国际版吧。

解锁

访问http://www.miui.com/unlock/index.html申请解锁，下载解锁软件并按照说明对手机进行解锁。

下载小米助手

访问http://zhushou.xiaomi.com/下载小米助手。

安装手机驱动

安装小米助手之后，打开小米助手，先在手机正常开机的情况下，用 USB 数据线连接一下，不用管是否能用小米助手连接成功，主要是这个连接会把手机的驱动安装上。

刷机

国内版刷国际版要清数据，所以在刷机之前请自行备份数据，而且要备份到手机以外的其他设备，否则会丢失数据。

驱动安装完成之后，拔掉 USB 数据线，将手机关机。然后同时按住音量-和电源两个按钮，直到机器进入Fastboot界面，再连接 USB 数据线，点击小米助手右下角“刷机”按钮，选择“Fastboot 刷机模式”。

在弹出的窗口中，区域选择“国际版”，点击其后的“检测刷机包”按钮，在刷出来的 ROM 里选择开发版，并勾选“刷机时清除手机数据”。然后点击下载刷机包开始刷机（过程有点长，需要先下载再刷机，慢慢等待，注意别让电脑休眠），直到手机自动重启时拔掉 USB 数据线。

刷机完成后进入系统做简要设置，使手机能联网正常使用。

开启开发者模式及 USB 调试

进入设置->关于手机，连续点击MIUI 版本打开开发者模式，返回设置，进入更多设置里的开发者选项，开启开发者选项，开启 USB 调试。

Root 手机

下载第三方 Recovery

访问https://twrp.me/，从Devices里选择手机相应的品牌和型号，下载对应版本的 TWRP 镜像。红米3s 直接访问https://twrp.me/xiaomi/xiaomiredmi3s.html，点击链接“Primary (Americas)”下载镜像。

下载 Google 官方工具包

https://dl.google.com/android/repository/platform-tools-latest-darwin.zip
https://dl.google.com/android/repository/platform-tools-latest-linux.zip
https://dl.google.com/android/repository/platform-tools-latest-windows.zip

下载 Magisk 框架

用手机访问下载或者在电脑上下载完成后将这三个文件传到手机内。

访问https://www.didgeridoohan.com/magisk/MagiskInstallation下载其中的Magisk zip、Magisk Manager apk和Uninstall zip。

刷第三方 Recovery

  • 解压Google 官方工具包，并在解压后的目录内运行命令行工具（Windows 系统在资源管理器地址栏输入“cmd”三个字母按回车键，会弹出命令行窗口）
  • 将第三方 Recovery 复制到Google 官方工具包解压后的目录内，并重命名为简短名称，比如“twrp”
  • 将手机进入Fastboot界面（前边有说进入的方法）
  • 连接 USB 数据线，并在命令行里输入fastboot devices，如果回显类似“f434e047d030 fastboot”文字，表名已经正常连接。
  • 在命令行输入刷 Recovery 命令fastboot flash recovery twrp.img并回车（命令中的twrp是之前步骤中重命名的第三方 Recovery 镜像名称加后缀）。出现“Finished.”字样就是刷入成功。
  • 断开 USB 数据线，按住电源键，直到手机重启出现 MI 字 LOGO 时，同时按住音量++电源键，直到进入 TWRP 界面。

刷入 Magisk

  • 在 TWRP 界面的“select langage”里选择“Chinse simplified”，界面语言会切换成简体中文，向右滑动界面底部的箭头，允许修改。
  • 点击进入 Recovery 模式
  • 点击安装，通过文件选择窗口选择下载的“Magisk zip”文件，然后确认安装。
  • 安装完成后点击重启手机
  • 进入系统后用文件管理器找到“Magisk Manager apk”并安装。

安装和谐软件

  • 访问下载地址
  • 下载适用于手机的最新版本应用[红米3s选择arm64对应的版本]
  • 将下载的应用安装到手机上，打开应用，点击右下方纸飞机，等待纸飞机圆框变成绿色然后返回桌面。

安装映射相关应用

  • 访问https://github.com/slashmax/AAMirror/tree/master/apk下载并安装aa_mirror.apk和aa_unlock.apk两个应用
  • 打开应用aa_unlock.apk，同意授予它永久 root 权限
  • 打开 Play 商店，更新 Google 服务框架，下载 Android Auto、高德地图等。

设置

aa_mirror 设置

打开Android Auto应用，选择Screen settings。

  • 开启Overwrite screen brightness，Screen brightness value设为100（待映射成功后按需调整）。
  • Screen rotation选择Force
  • Rotation degree选择90
  • Immersive mode选择Full immersive

Android Auto 设置

打开Android Auto，点击汉堡图标，选择About，在关于 Andorid Auto标题栏连点10下打开开发者模式，然后点击标题栏最右侧的三个点，选择开发者设置，划到最底部，勾选未知来源，然后退出应用。

连接汽车

用 USB 数据线将手机连接到汽车中控下方的 USB 口，看到手机和中控屏上的提示，都选择同意，然后慢慢探索并享受 Android 映射吧！

异常情况解决

解决使用中遇到的一些异常情况及解决办法，遇到问题持续更新。

高德停止导航

  • 使用高德地图在（进出隧道、车库、收费站）等有光线明暗变化时，会出现停止导航并返回导航主界面的情况。解决办法是关闭手机自动亮度。
